Dead Wheel Odometry Setup Guide

Hardware Setup:

  • Mount three dead wheels perpendicular to each other (two parallel, one perpendicular)
  • Connect encoders to the REV Control Hub:
    • Left encoder: Port 0
    • Right encoder: Port 1
    • Center encoder: Port 2
  • Ensure wheels maintain constant contact with the ground

Software Configuration:

  • Add the following to your build.dependencies.gradle: 
    implementation 'com.acmerobotics.roadrunner:core:0.5.5'
  • Configure your robot's drive constants in DriveConstants.java
  • Run the training sequences in order:
    1. Forward/Backward Test
    2. Lateral Movement Test
    3. Angular Velocity Test
    4. Full Calibration Test
